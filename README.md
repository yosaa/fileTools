# 朵拉工具箱

一个功能强大的Windows桌面图片处理工具箱，具有美观的界面和丰富的功能。

## 功能特点

- 🎨 **奶白色主题** - 温馨的奶白色界面设计
- ✨ **毛玻璃特效** - Windows 11风格的毛玻璃效果
- 🎯 **圆角设计** - 现代化的圆角界面元素
- 🚀 **卡通按钮** - 可爱风格的按钮设计
- 📱 **响应式布局** - 适配不同屏幕尺寸

## 主要功能

1. **首页** - 欢迎界面和功能说明
2. **一键分类产品** - 智能识别并分类图片文件
3. **一键裁切图片** - 批量裁切图片到指定尺寸
4. **更多功能** - 持续更新中...

## 项目结构

```
DoraTools/
├── main.py              # 主应用程序
├── image_classifier.py  # 图片分类模块
├── image_cropper.py     # 图片裁切模块
├── requirements.txt     # 依赖包列表
├── run.bat             # Windows运行脚本
└── README.md           # 项目说明
```

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

双击 `run.bat` 文件，或者在命令行中运行：

```bash
python main.py
```

## 使用方法

1. **启动应用** - 双击运行 `run.bat`
2. **选择功能** - 点击左侧的功能按钮
3. **选择路径** - 点击"浏览"按钮选择图片文件夹
4. **开始处理** - 点击"开始处理"按钮
5. **查看结果** - 处理完成后会显示处理数量

## 开发说明

### 技术栈

- **Python 3.7+** - 主要编程语言
- **tkinter** - GUI框架
- **PIL/Pillow** - 图片处理库

### 扩展功能

要添加新功能，请按照以下步骤：

1. 在 `main.py` 中添加新的功能按钮
2. 创建对应的处理模块（参考 `image_classifier.py` 或 `image_cropper.py`）
3. 在 `process_images` 方法中添加新功能的调用

## 注意事项

- 确保图片文件夹路径正确
- 处理大量图片时可能需要一些时间
- 裁切功能会创建新的文件夹保存结果
